【1】各種ボタンの説明
 ・「基本情報」タブの登録ボタンを機能させるには、以下の必須入力欄を埋める必要がある
 　（登録ボタンからDBにinsert処理されるのも、いまは以下３項目のみが対象）
 　>出社先ビル名
 　>出社日付
 　>体温(記録時)

 ・「行動履歴詳細」タブの複製ボタン、編集ボタン、登録ボタンはまだ処理未実装
 　※押してもなにもしない

【2】行動履歴詳細タブ中の表の説明
　・入力フォームに適当な内容を入力して追加ボタンを押すと、下の表に反映される
　・表中のチェックボックスを選択した状態で削除ボタンを押すと、その行が削除される


■直近でやりたいこと、現時点での課題（おまけ）

　2. 複製ボタンと編集ボタンをそれぞれ機能させたい
